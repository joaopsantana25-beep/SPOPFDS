CD: change Directory (mudar diretório)

Usado para navegar entre diretórios no terminal linux

Seu principal parametro é o nome do diretório para o qual se deseja navegar

Alguns caracteres tem funções especiais associadas ao comando CD

Como por exemplo, o caracter .. que é utilizado para acessar o diretório pai do diretório atual

  • cd .. : Acessa o diretório pai do diretório atual
  • cd ~ : Acessa o diretório home do usuário
  • cd - : Acessa o diretório anterior
  • cd / : Acessa o diretório raiz do sistema

MKDIR: make directory (criar diretório)

Usado para criar novos diretórios no sistema Linux

Seu principal parâmetro é o nome do diretório que se deseja criar

Pode incluir opções para criar múltiplos diretórios ou estruturas completas

  • mkdir nome : Cria um diretório com o nome especificado
  • mkdir dir1 dir2 : Cria múltiplos diretórios
  • mkdir -p dir/subdir : Cria diretórios aninhados

CLEAR: limpar terminal

Usado para limpar a tela do terminal

Não possui parâmetros obrigatórios

Apenas remove o conteúdo visível da tela

  • clear : Limpa o terminal

MV: move (mover/renomear arquivos)

Usado para mover ou renomear arquivos e diretórios

Seus principais parâmetros são o arquivo de origem e o destino

Pode ser usado tanto para mover quanto para renomear

  • mv arquivo destino : Move o arquivo para outro local
  • mv nome_antigo nome_novo : Renomeia o arquivo
  • mv -i : Solicita confirmação antes de sobrescrever

CP: copy (copiar arquivos)

Usado para copiar arquivos e diretórios

Seus principais parâmetros são a origem e o destino

Possui opções para cópia recursiva

  • cp arquivo destino : Copia um arquivo
  • cp -r pasta destino : Copia diretórios recursivamente
  • cp -i : Solicita confirmação antes de sobrescrever

NANO: editor de texto

Usado para editar arquivos de texto diretamente no terminal

Seu principal parâmetro é o nome do arquivo

Possui atalhos de teclado para salvar e sair

  • nano arquivo.txt : Abre ou cria um arquivo
  • Ctrl + O : Salva o arquivo
  • Ctrl + X : Sai do editor

PING: testar conectividade de rede

Usado para verificar a comunicação com outro dispositivo na rede

Seu principal parâmetro é o endereço IP ou domínio

Envia pacotes e mede o tempo de resposta

  • ping google.com : Testa conexão com um site
  • ping -c 4 endereço : Limita o número de pacotes enviados

FIND: buscar arquivos

Usado para localizar arquivos e diretórios no sistema

Seu principal parâmetro é o caminho e o nome do arquivo

Permite buscas avançadas com filtros

  • find . -name arquivo.txt : Busca pelo nome
  • find / -type d : Busca diretórios
  • find . -size +10M : Busca arquivos maiores que 10MB

CAT: concatenar/exibir arquivos

Usado para exibir o conteúdo de arquivos no terminal

Também pode ser usado para concatenar arquivos

É um comando simples e muito utilizado

  • cat arquivo.txt : Exibe o conteúdo
  • cat arq1 arq2 : Junta conteúdos
  • cat > arquivo : Cria/edita um arquivo

CHMOD: change mode (alterar permissões)

Usado para modificar as permissões de arquivos e diretórios no Linux

Seu principal parâmetro define quais permissões serão atribuídas

As permissões podem ser representadas por letras (r, w, x) ou números (chmod numérico)

  • chmod 777 arquivo : Concede todas as permissões (leitura, escrita e execução)
  • chmod 755 arquivo : Permissão total para o dono e leitura/execução para outros
  • chmod +x arquivo : Adiciona permissão de execução
  • chmod -w arquivo : Remove permissão de escrita
  • chmod u+x arquivo : Adiciona execução para o dono (user)

SU: switch user (trocar usuário)

Usado para alternar entre usuários no sistema Linux

Seu principal parâmetro é o nome do usuário que se deseja acessar

Quando utilizado sem parâmetros, tenta acessar o usuário root

  • su : Alterna para o usuário root
  • su usuario : Alterna para outro usuário
  • su - : Carrega o ambiente completo do usuário (login)
  • su - usuario : Alterna para outro usuário com ambiente completo

IP: gerenciar rede

Usado para visualizar e configurar interfaces de rede

Substitui comandos antigos como ifconfig

Possui diversos subcomandos para diferentes funções

  • ip a : Mostra endereços IP
  • ip link : Mostra interfaces de rede
  • ip route : Exibe rotas de rede
